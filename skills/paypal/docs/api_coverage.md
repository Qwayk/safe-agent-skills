# API coverage (endpoints -> CLI)

This is the main reference for shipped runtime commands in this slice.

## Source summary

- Provider: PayPal REST APIs (official resources only)
- API base URL: `https://api-m.sandbox.paypal.com` (production: `https://api-m.paypal.com`)
- Auth method: OAuth 2.0 bearer token
- Shipped command count: 133
- Notes: commands flagged with `partner-gated`, `account-gated`, or `live-unverified` in notes need feature/account checks before use.

## Out of scope

- `orders server.callback` is an inbound callback contract and is intentionally excluded from runtime CLI commands.
