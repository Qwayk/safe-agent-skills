# What you can do with YouTube

YouTube work usually starts with a channel, video, playlist, caption, comment, or upload question that needs official data before someone edits a public channel.

This skill can help with channel research, video audits, caption downloads, playlist cleanup, metadata planning, and careful upload or update plans. The safest pattern is simple: read the channel first, explain what was found, then prepare a reviewed plan before anything changes.

If you need setup first, start with [Connect your YouTube account](onboarding.md). If you need exact commands, use [Quickstart](quickstart.md) and [Command reference](command_reference.md).

## Good first asks

- "Resolve this channel from a handle or URL, then show the latest uploads."
- "Search this channel for videos about this topic and tell me which ones need attention."
- "Export the public videos from this channel's uploads playlist so I can review them."
- "Check whether I have access to captions for these videos and download the tracks I can use."
- "Plan a metadata update from this spreadsheet, but stop before changing YouTube."

## Channel and video research

Good research jobs include:

- "Which recent videos are missing useful titles, descriptions, links, or tags?"
- "Which videos should be grouped into a playlist for this topic?"
- "Which older videos mention an outdated offer, URL, or product name?"
- "Which videos have captions and which ones still need them?"
- "Which uploads look most relevant for a landing page, help article, or campaign?"

The useful answer should include video IDs or links, titles, dates, and the reason each video was flagged.

## Publishing and metadata planning

YouTube changes are public and easy to get wrong, so the agent should help plan before it acts:

- "Draft a private upload plan for this file, title, description, tags, and playlist."
- "Preview title and description changes for these videos and show the diff."
- "Plan caption uploads or replacements and explain the approval needed."
- "Plan playlist creation or video ordering before applying it."
- "Prepare a comment reply or moderation action, but do not post it yet."

The agent should say clearly whether the next step is only a local export, a live read, or a real YouTube write.

## Caption and localization work

Captions are a good place to use this skill because the output often needs to become a file:

1. Find the videos that need caption review.
2. Download caption tracks the account can access.
3. Save them locally for editing, translation, or quality checks.
4. Prepare an upload or replacement plan only after the edited file is ready.

If caption access is not allowed for a video, the agent should say that plainly instead of pretending the track can be downloaded.

## What good output looks like

A useful YouTube answer should include:

- the channel, video, playlist, caption track, or comment target
- the live read or local file that supports the answer
- a short human summary of what matters
- any saved export path
- the approval gate for uploads, metadata updates, deletes, or other writes
- a clear stop if the channel, file, permission, or target cannot be identified safely

## Honest limits

The tool uses YouTube API permissions from the connected account. Some data depends on ownership, scopes, and YouTube's own rules. If the agent cannot prove access or identify the exact target, it should ask for the missing detail before it plans a change.
