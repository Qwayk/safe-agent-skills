# What you can do with Unsplash

Unsplash is useful when you need real photo records, creator context, collections, topics, statistics, and download planning before choosing images for content or campaigns.

Ask the agent for the image research work a writer, designer, marketer, or editor does before publishing: find strong photo options, build a consistent shortlist, compare creators or collections, export research, and plan approved downloads carefully.

## Good questions to ask

- "Can you find 20 photos for this topic and shortlist the best options?"
- "Can you build one hero image plus three supporting images with a consistent style?"
- "Which creator portfolio, topic, or collection fits this campaign?"
- "Can you compare these photo IDs and tell me which one works best for a homepage hero?"
- "Can you pull photo or user statistics before we choose final images?"
- "Can you export this search so I can review the same results later?"
- "Can you plan downloads for these approved photo IDs without downloading yet?"
- "Will this download overwrite a local file or trigger Unsplash download tracking?"

## Everyday work this helps with

### Photo research

The agent can search photos by topic, mood, product, campaign idea, creator, topic, or collection. The useful output is a shortlist with links and short notes, not a dump of every result.

### Visual set building

Ask for one hero image and supporting images with a consistent style. The agent should explain why the set works together and where each image might fit.

### Selection review

The agent can compare photo IDs, creator context, collections, topics, and statistics before a person chooses final images.

### Careful download planning

For approved photo IDs, the agent should preview the destination folder, overwrite risk, and download-tracking step before any download apply.

## What the agent should show you

- The search topic, collection, creator, photo IDs, or statistics target it used.
- A shortlist with links and short reasons.
- The difference between research/export work and approved download work.
- The destination path before any local file write.
- Whether the download would overwrite an existing file.
- A review plan before download tracking or local file writes.

## Good first path

Start with: "Check the Unsplash skill is configured, find 20 photos for my topic, and build a shortlist before we plan any downloads."

After that, ask one useful follow-up, such as: "Build a consistent hero-plus-supporting image set" or "Plan downloads for these approved photo IDs without downloading yet."
