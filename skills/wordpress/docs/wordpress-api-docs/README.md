# Vendored WordPress REST API notes (offline)

Last verified (UTC): 2026-01-27

This folder contains short, offline-friendly summaries of key WordPress REST API endpoints used by `wordpress-api-tool`.

Principles:
- Keep these docs **concise** (no large copied pages).
- Link to official docs for full parameter lists and schema details.
- Focus on “what matters” for safe CLI behavior: permissions, pagination, common filters, and response shape.

Reference files live in `docs/wordpress-api-docs/reference/`.
