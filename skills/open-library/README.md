# Open Library

Install slug: `open-library`

Use this skill when you want your AI agent to search public Open Library books, authors, works, editions, and subjects without any live account setup.

This tool reads public Open Library catalog data from a command line.
It is read-only, needs no auth, and only exposes named commands for the small API surface in [API coverage](docs/api_coverage.md).
Open Library asks developers to use these endpoints for real-time, low-volume use, not bulk downloading.
