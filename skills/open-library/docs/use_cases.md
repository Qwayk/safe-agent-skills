# Use cases

Use this page when you want ideas for real Open Library jobs to hand to your agent.
If you need setup first, start with [Use Open Library with no account](onboarding.md). If you need exact commands, use [Quickstart](quickstart.md) and [Command reference](command_reference.md).

You can read Open Library data for research and review.
No login or API key is needed.

Good first asks:

- "Find five books about Dune and show the most useful matches."
- "Open this author record and list the author's works."
- "Look up this ISBN and explain which edition it is."
- "Open this work ID and show the available editions."
- "Try a small subject search and tell me whether the results look useful."

Use `subjects get` for testing topical queries only.
It is marked experimental and is for exploration, not production workflows.
