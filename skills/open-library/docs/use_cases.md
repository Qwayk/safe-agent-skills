# Use cases

Use this page when you want ideas for real Open Library jobs to hand to your agent.
If you need setup first, start with [Connect your account](onboarding.md). If you need exact commands, use [Quickstart](quickstart.md) and [Command reference](command_reference.md).

You can read Open Library data for research and review.
No login or API key is needed.

Use cases:

- Find book candidates with `search books`.
- Find an author and check their profile with `authors get`.
- Expand work details and edition history with `works get` and `works editions list`.
- Resolve ISBN to catalog records with `isbn lookup`.
- List an author’s works with `authors works list`.

Use `subjects get` for testing topical queries only.
It is marked experimental and is for exploration, not production workflows.
