# What you can do with Google Search Console

Search Console work usually starts when someone needs Google's view of a site, not another SEO guess: which queries bring traffic, which pages changed, whether a URL is indexed, and whether sitemaps are healthy.
If you need setup first, start with [Connect your account](onboarding.md). If you need exact commands, use [Quickstart](quickstart.md) and [Command reference](command_reference.md).

This skill is strongest when the agent reads Search Console first, explains the pattern in plain English, and only then prepares a reviewed sitemap or site-property change.

## Good jobs to give the agent

### Weekly SEO checks

- "List the Search Console sites I can access and tell me which ones look relevant."
- "Show the top queries and pages for this property over the last 28 days."
- "Compare query clicks, impressions, CTR, and position by page so I can see what changed."
- "Pull Search Analytics by country and device so I can see whether the drop is mobile, desktop, or one market."
- "Export the top pages and queries for a simple SEO handoff."

### URL and indexing review

- "Inspect this URL and explain what Google says about indexing."
- "Check whether this page is canonical, indexed, blocked, or has a mobile issue."
- "Inspect the URL before and after a page update so we know what Google currently sees."
- "Review a few important URLs and group them by issue type."

### Sitemap and property work

- "List submitted sitemaps and flag anything stale, broken, or surprising."
- "Check this sitemap before I submit it."
- "Preview submitting this sitemap so I can review it before anything changes."
- "Prepare a reviewed plan to delete a stale sitemap, but do not apply it yet."
- "Check whether this site property exists before we plan adding or removing it."

## What the agent should show you

- The exact property, URL, sitemap, date range, dimensions, and filters it used.
- A short summary of the top query, page, country, device, indexing, or sitemap pattern.
- A clear warning when Search Console access, property verification, or service-account access is missing.
- A dry-run plan before site add/delete or sitemap submit/delete.
- A receipt or clear refusal after any approved live change.

## Good first SEO path

Start by listing accessible sites, choose one property, run a 28-day query and page report, then inspect one important URL. That gives the agent enough context to avoid guessing.
