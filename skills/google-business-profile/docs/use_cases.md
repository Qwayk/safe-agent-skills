# Use cases

Google Business Profile is useful when local customers need the right information before they call, visit, book, or ask for directions.

Ask the agent for the work a local business owner, agency, or operations person checks again and again: hours, phones, addresses, categories, reviews, photos, verification, and anything Google changed on the listing.

## Good questions to ask

- "Which locations have missing basics?"
- "Are the hours, phone number, website, address, categories, and attributes correct?"
- "Did Google suggest or apply any changes that we need to review?"
- "Which locations are not verified, suspended, or stuck?"
- "What reviews came in recently, and which ones need a reply?"
- "Do the photos, services, booking links, and call settings look complete?"
- "Which locations are getting more calls, directions, or website clicks?"
- "Can you prepare a careful update for this location and show me what would change?"

## Everyday work this helps with

### Location cleanup

The agent can list accounts and locations, then point out missing or suspicious details. This helps before a new client handoff, a local SEO cleanup, or a brand consistency check across many stores.

### Review response support

Ask for recent reviews, common themes, and a suggested reply plan. The agent should keep the business name, location, rating, and review text clear so a person can approve the right response.

### Verification and access checks

The agent can check verification status, invitations, admins, and transfer details. This is useful when a location cannot publish changes or the wrong person owns the profile.

### Profile change review

For hours, categories, attributes, media, lodging details, place action links, or notification settings, the agent should show the current listing details first and then the proposed update.

## What the agent should show you

- The account and location it is working on.
- The current public-facing details before it suggests changes.
- Recent reviews and the exact review target before any reply work.
- Verification, admin, invitation, or transfer status in plain English.
- Performance numbers with the date range clearly named.
- A review plan before any profile, media, review, access, or verification change.

## Good first path

Start with: "Check the Google Business Profile connection, list my accounts and locations, and show me which locations have missing basics or need review."

After that, ask one useful follow-up, such as: "Show me recent reviews that need attention" or "Check this location's hours, categories, attributes, and Google-updated fields."
