# Use cases

Use this page when you want ideas for real Pipedrive jobs to hand to your agent.
If you need setup first, start with [Connect your account](onboarding.md). If you need exact commands, use [Quickstart](quickstart.md) and [Command reference](command_reference.md).

## Why this is useful

- It stays read-only by product choice, so it cannot change CRM data by mistake.
- It gives deterministic JSON output that is easy to audit.
- It covers the documented public REST read surface without using a raw request bridge.

This is for review and read work only. For any write changes, use another tool.
