# Pipedrive

Install slug: `pipedrive`

Use this skill when you want your AI agent to review deals, people, organizations, activities, products, and other Pipedrive CRM data without changing anything live.

## Read-only by product choice

- You can read, search, and compare Pipedrive records.
- You cannot create, update, or delete anything with this skill.
- Write-style actions are excluded on purpose and are not in the shipped command map.
